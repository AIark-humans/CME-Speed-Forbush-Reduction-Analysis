# CME-Speed-Forbush-Reduction-Analysis
This repository includes the following:
- All raw data utilized
- All data manipulation programs
- Results including: Graphs, Correlation Coefficient, Error, and Fractional Error

Only a small portion of the sorted coronal mass ejection data is provided. Use scripts/cleaner.py to generate the rest.

Data From:
Nasa CDAW Database
Neutron Counting Station Oulu Finland

Works Cited:
NASA (2026). SOHO LASCO CME CATALOG (2). NASA. Available from: https://cdaw.gsfc.nasa.gov/CME_list/UNIVERSAL_ver2/text_ver/
Cosmic Ray Station of the University of Oulu/ Sodankyla Geophysical Observatory (2026). Oulu NM. University of Oulu. Available from: https://cosmicrays.oulu.fi/
